# python-mysql-example

A small example to show how to use **mysql** in python.

## Usage

```shell
./main.py   # OR    python main.py
```

Enter a string (name) and a number (revenue):

```shell
Customer name: foo
Customer revenue: 123

READ:   []
CREATE
READ:   [('foo', None)]
UPDATE
READ:   [('foo', Decimal('123'))]
DELETE
READ:   []
```
